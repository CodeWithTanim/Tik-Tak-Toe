## **🎮 How to Play?**
### **Single-Player**  
1. Select "Play vs AI" 🤖  
2. Choose difficulty:  
   - **Easy**: Random moves  
   - **Normal**: Strategic defense  
   - **Hard**: Perfect AI (impossible to win)  
3. Play as ❌ against AI (⭕)  

### **Multiplayer**  
1. Enter player names 👤👤  
2. Alternate turns placing ❌ and ⭕  
3. First to 3 in a row wins!
